# dotfiles

- .zshrc

## Colors

```zsh
fg="\e[38;5;"
bg="\e[48;5;"
reset="\e[0m"
grey="0m"
red="1m"
green="2m"
yellow="3m"
blue="4m"
pink="5m"
green="6m"
white="7m"
```

- `${fg}` applies foreground colors when followed by the color code
- `${bg}` applies background colors when followed by the color code
